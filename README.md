## Shopping List App

- Publicly available on: https://shoppinglistapp-3nwt.onrender.com/
- Backend server available on: https://shopping-json-server.onrender.com/

#### How To Clone

1. Clone the project from this repository and open it on your desired IDE (prefered: VSCode)

```bash
git clone https://github.com/msizi007/ShoppingListApp.git
```

2. Install all node packages using `npm install | npm install`

```bash
npm install
```

3. Run npm project using `npm run dev`

```bash
npm run dev
```

#### Tech Stack

- ReactJS
- Bootstrap (CSS)
